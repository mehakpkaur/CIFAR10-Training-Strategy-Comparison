# Deep Learning Training Strategy Analysis on CIFAR-10

## Overview
This project compares the performance of different deep learning architectures under multiple training strategies using the CIFAR-10 image classification dataset.

## Architectures Used
- VGG-style CNN
- ResNet-style CNN
- MobileNet-style CNN

## Training Strategies
- Full Dataset Training
- Progressive Dataset Revelation
- Curriculum Learning

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset
CIFAR-10 dataset containing 60,000 images across 10 classes.

## Results
- Best Accuracy: 91.64%
- Best Model: ResNet with Full Dataset Training

## Key Findings
- Full dataset training achieved highest accuracy.
- Progressive and curriculum learning reduced training time.
- ResNet consistently outperformed other architectures.
